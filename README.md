# Bedlam-Cube

## Running

```bash
./cube
```

## Installation

1. Clone repository

```bash
git clone https://github.com/Bungogood/Bedlam-Cube.git
```

2. Create .obj directory

```bash
cd Bedlam-Cube
mkdir .obj
```

3. Make project

```bash
make
```
